== Description ==

WPRunway for Divi enhance your website with powerfuls and useful new modules and features.

[Check the demos and documentation](https://www.wprunway.com)

Note: [Divi](https://www.elegantthemes.com/gallery/divi/) Website builder is required for this plugin.

Runway for Divi provides new modules:

* [Alert](https://www.wprunway.com/module/alert) – Display alert messages to your visitors.
* [Note](https://www.wprunway.com/module/note) – Show notes in your pages or posts.
* [Reading time](https://www.wprunway.com/module/reading-time) – Let your visitors know how long it takes to read a post or page.
* [Tag](https://www.wprunway.com/module/tag) – Insert a tag at the biginning, midle or end of a sentence.

== Installation ==

> Note: This plugin works with Divi. Make sure you have [Divi Theme](https://www.elegantthemes.com/) installed and activated.

1. Upload the plugin files to the `/wp-content/plugins/exclusivedivimodules` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Exclusive Divi option, will appear in the dashboard.

== Frequently Asked Questions ==

= Can we use modules in theme builder? =

Yes, you can use every modules inside Divi theme builder.

= How to disable specific modules? =

You can disable any module from your WordPress Dashboard > Runway for Divi > Modules tab.
